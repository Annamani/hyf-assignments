# Week-1 
This folder contains exercises focused on essential JavaScript methods used for working with arrays, objects. 
## Things learned
### 1. forEach()
- Used to iterate over each element in an array.
- Does not return a new array.
### 2. map()
- Creates a new array by applying a function to every element in the original array.
- Size of the array should be same as original Array.

### 3.filter()
- Creates a new array containing elements that pass a condition.
- This method does not change the original array.
- Size of the array may be different to Original array.

### 4. Arrow Functions
- A shorter syntax for writing functions.

### 5. reduce()
- Reduces an array to a single value.
- This method does not change the original array.

### 6. Object.entries() 
- Converts an object into an array of [key, value] pairs.
- Useful when iterating through object properties.

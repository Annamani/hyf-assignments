## Part B: Ethics and Risks
# Over-reliance on AI may lead to lack of Deep learning
When I enable Copilot in my IDE, its suggestions can distract my thinking, and blindly following them may introduce errors or break other parts of my code.
Mitigation: I will use AI as a guide, carefully review suggestions, and ensure I fully understand the code before implementing it.

# Confusing Multiple Answers
AI can provide multiple solutions for the same question, which can be confusing when learning a new topic.
Mitigation: I will compare different suggestions carefully, research the reasoning behind each, and choose or adapt the solution that best fits my understanding and project requirements.

# Risks using AI code directly without understanding
AI-generated code can sometimes contain vulnerabilities or unsafe practices that could compromise the project.
Mitigation: I make sure to thoroughly check, test, and adjust AI-suggested code before implementing it according to my requirement.
